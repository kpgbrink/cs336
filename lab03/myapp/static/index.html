Lab 3

Answers to lab 3 exercises:

  • 3.1
    1. Compare and contrast the purposes of Node.js and of Express.js.
      Node.js is generally a server side platform for building server-side i/o application using javascript while express.js is a framework that uses node.js for web-applications. Express is built on top of Node so Express adds more features.
    2. Explain the purpose of the package.json file.
      package.json holds meta data relevant to the project. It shows what dependencies there are and it enables things to be published with npm.
  • 3.2
    1. Explain the sense in which these files are static.
      The files are sent directly to the client. Express.js just outputs the files as they are in the file system.
    2. Determine whether the server continues to serve up the hello-world message and explain why this is the case.
      Yes, Express will match what is being asked for. When localhost:3000/ is requested the '/' matches the app.get('/' and it will run the function that sends Hello World! app.get is matched because it is before the app.use.
